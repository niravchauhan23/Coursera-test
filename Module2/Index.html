Componets of Front end devlopment

1. HTML

HTML (Hypertext Markup Language): HTML is the standard markup language used to create web pages. It provides the structure and content of a webpage by using a system of tags and attributes to define elements such as headings, paragraphs, images, links, and more.

2. CSS

CSS (Cascading Style Sheets): CSS is a style sheet language used to control the presentation and layout of HTML documents. It allows web developers to specify the appearance of elements on a webpage, including aspects such as colors, fonts, spacing, and positioning. CSS works by targeting HTML elements and applying styles to them using selectors and declarations.

3. JavaScript

JavaScript: JavaScript is a high-level programming language that is commonly used to add interactivity and dynamic behavior to web pages. It can manipulate the content and structure of HTML documents, respond to user actions such as clicks and input, and communicate with web servers to update content without reloading the page. JavaScript is often used in conjunction with HTML and CSS to create interactive and engaging web experiences.
